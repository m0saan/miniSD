# miniSD

_miniSD_ is a pedagogical project aimed at demonstrating the principles and working of stable diffusion algorithms in a simplified, intuitive manner.

## Setup

### Requirements

- Python 3.x

### Installation

```bash
git clone https://github.com/m0saan/miniSD.git
cd miniSD
poetry install
```

### Run

```bash
python sample.py
```
